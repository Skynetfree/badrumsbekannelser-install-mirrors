Install mirror pie
==================

- Install Raspbian with NOOBS
- Download these files
- Open Terminal
- Run first.sh (Fill in the uuid for the family in the last step)
  - `sudo ./first.sh`
- Reboot


- Connect to family wifi
- Open terminal
- Run second.sh
  - `sudo ./second.sh`

- The pi will reboot and open into Chromium with the family page
